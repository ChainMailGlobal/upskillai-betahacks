# Source Governance

The system should be text-first and provenance-aware.

## Priority Order

1. Manufacturer documentation
2. Owner manuals
3. Official safety and code sources
4. Trusted text-based trade publications
5. YouTube instructional videos as visual reference only

## What Text Sources Are For

Use text sources to drive:

- step order
- tool and material requirements
- safety warnings
- torque, clearance, and measurement notes
- terminology
- permit and inspection guidance

## What YouTube References Are For

Use YouTube references only to inform:

- camera angles
- framing and pacing
- hand motion timing
- spatial anchor placement
- scene transitions

Do not treat third-party YouTube footage as reusable production media unless
you have explicit permission to do so.

## Output Policy

Generated lesson clips should be:

- newly rendered with Seedance
- based on abstracted prompts
- backed by text citations
- safe to review step-by-step

They should not be:

- frame-copied from a creator video
- rehosted source footage
- close mimicry of a creator's identity or voice

## Required Metadata Per Lesson

Each lesson in the reference library should keep:

- trade
- risk class
- expert gate flag
- primary text sources
- YouTube visual references
- backend enrichments
- overlay beats

## Assist Mode Rule

If the system is operating live on a real job:

- text sources remain authoritative
- visual model guesses are advisory
- remote expert decisions override AI suggestions
- code and permit statements must be jurisdiction-scoped
